Acerca del proyecto EnviroPro

EnviroPro es una aplicación web desarrollada con Django para la gestión y análisis de datos ambientales procedentes de sensores IoT.

Objetivo

El objetivo del sistema es monitorizar variables ambientales como la humedad del suelo, la temperatura, el estado de la batería y la energía del panel solar, con el fin de detectar anomalías y generar alertas automáticas.

Funcionalidades
  • Importación de datos desde archivos CSV
  • Visualización de registros históricos
  • Generación automática de alertas
  • Sistema de recomendaciones asociadas
  • Dashboard con métricas y gráficos interactivos
  • Filtros, paginación y gestión de datos
Tecnologías utilizadas
  • Django
  • Pandas
  • Bootstrap
  • Chart.js
Autores

Asignaturas
